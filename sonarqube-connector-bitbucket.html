SonarQube Connector for Bitbucket

Conecta SonarQube Server o SonarQube Cloud con tu instancia de Bitbucket.

Toda la información de la calidad del código a tu alcance. SonarQube Connector trae tu modelo de calidad de código fuente a tu repositorio de Bitbucket, incluido el estado del umbral de calidad.

Pruébala gratis  

¿Por qué utilizar esta app?

Conoce aquí las principales características de nuestro conector de SonarQube for Bitbucket

Quality Gate

Estado y condiciones del umbral de calidad directamente en el repositorio de Bitbucket.

Más de una instancia

Es posible configurar diferentes instancias de SonarQube Server o SonarQube Cloud que se utilizarán desde los diferentes repositorios de código de Bitbucket.

Consulta tus métricas

Puede ver las métricas de SonarQube Server o SonarQube Cloud más importantes dentro de una tarjeta en la parte superior de la página de origen del repositorio de Bitbucket.

Integra la solución de Sonar con la de Atlassian

Consulta la información de la calidad del código desde Bitbucket

Atlassian y SonarSource juntos.

SonarQube Connector for Bitbucket integra las medidas de SonarQube Server o SonarQube Cloud en la instancia de Bitbucket.

Con este plugin, proporcionamos toda la información en Bitbucket, para que los usuarios no tengan que cambiar su contexto.

Sin salir de Bitbucket, podrán ver si el proyecto cumple o no cumple con las condiciones de calidad establecidas en el umbral de calidad (Quality Gate), incluido el estado de la puerta de calidad y las métricas de calidad del código más importantes:

  • Fiabilidad
  • Seguridad
  • Mantenibilidad
  • Revisión de seguridad
  • Densidad de duplicaciones
  • Líneas de código (ncloc)
  • Cobertura de código

¡La calidad del código fuente siempre bajo control!
