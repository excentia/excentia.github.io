¿Qué es Jira Software?


Jira Software es una aplicación de gestión de proyectos diseñada por Atlassian para planificar, supervisar, informar y automatizar las tareas de los equipos ágiles. Con Jira Software los equipos DevOps centralizarán en un único lugar la información de todo el ciclo de vida de un producto, desde el seguimiento del proceso de desarrollo, control de versiones, hasta el histórico de mejoras del producto. Todo ello fomentando la colaboración del equipo en tiempo real y el trabajo ágil.

Funcionalidades


Los equipos de trabajo pueden utilizar Jira Software para asignar y crear tareas, realizar un seguimiento de las mismas y colaborar en tiempo real. Además, cuenta con las siguientes funcionalidades que te desglosamos a continuacion:

Filtros
personalizados


Jira Software permite crear filtros personalizados con los resultados de una búsqueda realizada por Jira Query Language (JQL). Los resultados obtenidos de las consultas se podrán guardar para que las puedas utilizar como filtros y vistas, incluidos los tableros.

Integraciones para desarrolladores


Esta solución de Atlassian se integra con herramientas para desarrolladores para obtener una trazabilidad completa. Ejemplo de ello son Bitbucket, Gitlab, GitHub y herramientas de gestión documental como Confluence u otras soluciones de supervisión de código.

Ampliar
funcionalidades


El poder de Jira Software se puede ampliar gracias a la integración con más de 3.000 aplicaciones que están disponibles en el Marketplace de Atlassian.

Visualización
rápida


Jira Software permite gestionar las tareas con los tableros scrum y Kanban para facilitar la visualización del estado cada tarea y maximizar la eficiencia de los equipos. Esta solución recoge los datos de los proyectos y los plasma en vistas agregadas para facilitar la toma de decisiones

Aplicación
móvil


Haz que el trabajo avance con tu dispositivo móvil. La app de Jira para iOS y Android están disponibles así como la versión para navegadores web. Ambas plataformas cuentas con funcionalidades específicas según el tipo de dispositivo desde el que se está utilizando.

API
potentes


Automatiza los procesos con el sólido conjunto de API de Jira. Esta funcionalidad permite que Jira se puda integrar con una amplía variedad de servicios con su API.

Jira Software Cloud

Jira Software Cloud está equipado con todo lo que los grandes equipos de desarrollo ágil en crecimiento necesitan para entregar proyectos avanzados desde el primer día y sin poner en peligro la fiabilidad, la integridad de los datos o los requisitos de conformidad.

  • Controles de administración y seguridad - Combina Jira Software con Atlassian Access para disfrutar de visibilidad y controles mejorados con el SSO de SAML, el aprovisionamiento de usuarios (SCIM), los registros de auditorías y mucho más.
  • 99,9% de tiempo de actividad garantizado - SLA con 99,9 % de tiempo de actividad con apoyo financiero para que tu equipo pueda centrarse en el trabajo que resulta crítico para tu empresa sin preocuparse por el tiempo de inactividad.
  • Cifrado de los datos - Haz que el trabajo avance desde cualquier lugar con Jira, gracias a numerosos controles de seguridad, como el cifrado en tránsito y en reposo para Jira para dispositivos móviles.
  • Conformidad - Trabajamos con certificaciones SOC2, SOC3, ISO 27001, ISO 27018 y PCI DSS, entre otras, de forma que puedas verificar que la empresa y los datos de los clientes permanecen protegidos y conformes a las normativas.
  • Almacenamiento ilimitado - Todo el espacio que necesitas para gestionar cualquier tipo de proyecto en Jira Software.
  • Privacidad - Atlassian ha invertido mucho en el RGPD y en el Escudo de Privacidad para garantizar que tus datos se gestionen con sumo cuidado y de conformidad con las obligaciones de privacidad de datos en todo el mundo.
  • Soporte Premium - Todo seguirá funcionando a la perfección gracias al soporte Premium ininterrumpido y a los tiempos de respuesta de una hora en el caso de incidencias críticas.

Jira Software Data Center

Jira Software Data Center es una solución autogestionada que te proporciona opciones de configuración para personalizar Jira para las necesidades específicas de tu equipo, así como herramientas administrativas avanzadas para gestionar el crecimiento.

Las actualizaciones sin tiempo de inactividad y la flexibilidad de implementación con proveedores de IaaS como AWS y Azure proporcionan acceso ininterrumpido y escalado elástico para satisfacer las demandas de los usuarios.

  • Seguridad - Aumenta la seguridad y agiliza el inicio de sesión de los usuarios con el soporte integrado de inicio de sesión único de SAML. Elige entre una amplia selección de proveedores de identidades populares.
  • Rendimiento a escala - Evita la degradación del rendimiento Cada nodo del clúster de tu Data Center aumenta la capacidad de usuarios simultáneos y proporciona una experiencia de usuario constante bajo cargas elevadas.
  • Herramientas administrativas para el crecimiento - Gestiona el crecimiento y aumenta el rendimiento y la legibilidad de Jira con herramientas administrativas como el archivado de proyectos y el optimizador de campos personalizados.
  • Alta disponibilidad - La agrupación en clústeres activo-activo también les ofrece a tus equipos acceso ininterrumpido durante las actualizaciones y reindexaciones de búsqueda.
  • Actualizaciones sin interrupciones - Implementa la versión más reciente de Jira Software cuando quieras y donde quieras. Despídete de las interrupciones programadas y vuelve a disfrutar de las noches y los fines de semana.
  • Carga distribuida - Distribuye la carga de forma inteligente en tu clúster para conseguir servicios de la mayor calidad. Dedica nodos a equipos concretos, tráfico de API o tareas, y dirige el tráfico restante a otros.
  • Ofertas de soporte flexibles - Elige entre soporte Select, Priority o Premier y mantén tu sistema ejecutándose sin problemas gracias al acceso a los equipos de soporte altamente cualificados y a los rápidos tiempos de respuesta.
