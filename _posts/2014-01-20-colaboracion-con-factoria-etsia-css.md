<b>Factoría ETSIA</b> en colaboración con <b>excentia</b> ha desarrollado CSS Validator que analiza recursos CSS localizando fallos de implementación, 
malas prácticas e incumplimiento de la normativa WCAG 1.0 y WCAG 2.0 para ayudar a la CSS que sea óptima y accesible. Las características de este 
nuevo producto son: Análisis a través de SonarQube de gramáticas CSS validando el correcto uso de Keywords, Propiedades y valores introducidos.


Más de 30 reglas de buenas prácticas relacionadas con la compresión, adaptabilidad y mal uso de Keywords.

Más de 30 reglas adicionales de accesibilidad WEB para ayudar a cumplir al usuario con la normativa WCAG 1.0 y WCAG 2.0. 
(Regla de contraste de colores, reglas de uso de Keywords necesarios para accesibilidad, reglas de validación de medidas, etc.). 

Más información y descarga en la página del producto.
