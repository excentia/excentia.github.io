La forma en que consumimos información no es la misma para todos. Mientras algunas personas prefieren leer texto, otras entienden mejor los datos **mediante gráficos, diagramas o presentaciones visuales**. Consciente de esta realidad, Atlassian ha incorporado Remix, una funcionalidad de Rovo que permite transformar contenido existente de Confluence en nuevos formatos visuales e interactivos con ayuda de inteligencia artificial.

En este artículo veremos **qué es Remix**, qué tipos de contenido puede generar y cómo utilizarlo paso a paso dentro de Confluence.

<h2>¿Qué es Remix en Confluence?</h2>

Remix es una función de [Atlassian Rovo](https://www.excentia.es/que-es-atlassian-rovo) para Confluence que **transforma contenido existente en gráficos, diagramas, infografías y visualizaciones** mediante inteligencia artificial. A partir de texto seleccionado en una página o un Live Doc, Remix puede generar automáticamente:

- Gráficos y charts.
- Diagramas.
- Infografías.
- Visualizaciones interactivas.

El objetivo es ofrecer **nuevas formas de representar la información** para adaptarse a diferentes necesidades, contextos y estilos de aprendizaje.

<h2>Casos de uso de Remix en Confluence</h2>

Remix en [Confluence](https://www.excentia.es/que-es-confluence-y-a-que-aporta-al-trabajo-remoto) puede utilizarse en distintos escenarios para transformar información textual en formatos visuales. Algunos ejemplos son:

- Convertir una lista de puntos en una infografía.
- Crear un organigrama a partir de una tabla con nombres y equipos.
- Generar diagramas editables a partir de información compleja.
- Visualizar datos de investigación mediante mapas interactivos.

Además, puedes **utilizar el chat de Rovo** para solicitar transformaciones específicas mediante instrucciones en lenguaje natural.

<h2>Cómo iniciar un Remix en Confluence</h2>

Si tienes permisos de edición sobre una página o <a href="https://www.youtube.com/watch?v=sXEoQCLj1bE" target="_blank" rel="nofollow">Live Doc</a>, existen tres formas de iniciar un Remix.

**Opción 1: Seleccionando texto**

- Abre una página o Live Doc en Confluence.
- Selecciona el contenido de texto que quieras transformar.
- En la barra flotante que aparece sobre la selección, haz clic en Remix.

Esta es la opción más rápida cuando deseas transformar un fragmento concreto de contenido.

**Opción 2: Desde la barra de herramientas de un elemento**

También puedes iniciar Remix desde la barra de herramientas de elementos existentes, como tablas.

- Selecciona el elemento.
- Abre su barra de herramientas contextual.
- Haz clic en Remix.

Esta opción resulta especialmente útil para trabajar con tablas y conjuntos de datos estructurados.

**Opción 3: Desde las acciones del bloque**

Otra alternativa consiste en utilizar las acciones disponibles en el lateral derecho de un bloque de contenido.

- Sitúa el cursor sobre un elemento de texto.
- Aparecerá el menú de acciones del bloque.
- Selecciona Remix.


<h2>Qué tipos de contenido puede generar Remix</h2>

Una vez iniciado el proceso, Remix muestra un menú con diferentes formatos de salida.

<img width="100%" src="/img/atlassian-products/confluence-remix-rovo-opciones.png" alt="Opciones Remix Rovo">
<br>

**Charts**

Cuando el contenido contiene datos numéricos, Remix puede generar gráficos automáticamente. Puedes elegir entre distintos tipos de visualización, como:

- Gráficos de líneas, de barras o circulares.
- Otros formatos adecuados para representar datos.

**Diagrams**

Los diagramas permiten representar visualmente las relaciones entre los distintos elementos del contenido. Remix puede generar:

- Diagramas de flujo.
- Mapas mentales.

Estos diagramas pueden añadirse al documento como Whiteboards incrustados.

**Infographics**

Las infografías ayudan a complementar el contenido mediante una representación más visual y narrativa. Actualmente pueden generarse en distintos estilos, incluyendo: Estilo moderno, minimalista o estilo boceto (sketch).

Una vez creada la infografía, puedes solicitar cambios a Rovo hasta obtener el resultado deseado y posteriormente insertarla o copiarla como imagen.

**Visualizations**

Las visualizaciones generan experiencias interactivas para explorar la información. Entre los ejemplos mencionados por Atlassian se encuentran:

- Líneas temporales.
- Mapas.
- Organigramas.
- Tarjetas interactivas (flip cards).

Este tipo de contenido resulta especialmente útil para presentar información compleja de forma más intuitiva.

Gracias a la integración de Remix con Rovo, los equipos pueden reutilizar contenido de páginas y Live Docs para presentar la información de una forma más visual, adaptada a diferentes audiencias y necesidades. Tanto si necesitas representar datos mediante gráficos como crear organigramas, mapas mentales o infografías, Remix ofrece una forma sencilla de generar nuevos formatos directamente dentro de Confluence.

<h2>Preguntas frecuentes sobre Remix en Confluence</h2>

**¿Remix modifica el contenido original?**
No. Esto significa que el contenido original nunca se sobrescribe ni se reemplaza. Cada Remix se crea como una capa adicional sobre la información existente, permitiendo explorar nuevas formas de visualizarla sin perder la fuente original.

**¿Puedo utilizar Remix en páginas y Live Docs?**
Sí. Remix puede utilizarse tanto en páginas de Confluence como en Live Docs.

**¿Necesito Rovo para utilizar Remix?**
Sí. Remix forma parte de las capacidades de Rovo y utiliza inteligencia artificial para generar nuevas representaciones a partir del contenido seleccionado.

**¿Qué datos necesito para crear un gráfico con Remix?**
Para generar un gráfico, el contenido seleccionado debe incluir datos numéricos. 

**¿Puedo crear organigramas con Remix?**
Sí. Las visualizaciones generadas por Remix pueden incluir organigramas creados a partir de tablas con nombres y datos de equipos.

**¿Puedo editar el resultado generado por Remix?**
Sí. Antes de insertarlo en la página, puedes revisar la vista previa, cambiar el estilo, seleccionar otro formato o solicitar modificaciones adicionales a Rovo.
