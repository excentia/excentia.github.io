What is SonarCloud?


It’s the SonarSource cloud solution that helps developers deliver clean code. SonarCloud performs automated code checks to quickly and effectively detect and troubleshoot issues. This solution has comprehensive coverage for languages and problems, following SonarSource’s Clean As You Code methodology.
