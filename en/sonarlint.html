What is SonarLint?


SonarLint is a free static code analysis extension that helps developers improve code quality directly from the integrated development environment (IDE). This solution detects and fixes code issues in real time while developers write and edit their code. This extension integrates seamlessly into a variety of development environments, including Eclipse, Visual Studio, IntelliJ IDEA and Visual Studio Code. Find and solve coding problems in real time, guaranteeing the quality of the coded and improving productivity.
