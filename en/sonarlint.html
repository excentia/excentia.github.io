SonarQube for ide (formerly SonarLint) is the extension for your development environment that allows you to detect problems in the code while you are writing it.

SonarLint works as a spell checker. It detects errors in the code while you are writing it.

SonarQube for ide works as a spell checker and detects and fixes problems in real time.

What is SonarQube for ide?


SonarQube for ide is a free static code analysis extension that helps developers improve the quality of their code directly from the Integrated development (IDE). This solution detects and corrects code problems in real time. This extension integrates seamlessly into a variety of development environments, as Eclipse, Visual Studio, IntelliJ IDEA and Visual Studio Code. Finds and resolves coding problems in real time, ensuring code quality and improving productivity.
