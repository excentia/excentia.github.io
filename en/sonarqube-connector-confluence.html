SonarCloud / SonarQube Connector for Confluence

Integrate your SonarQube or SonarCloud projects into your Confluence pages.

With this application you will be able to add and consult your metrics of SonarQube or SonarCloud without leaving Confluence and to consult the real state of quality of code quickly.

Why use this app?

SonarCloud / SonarQube Connector for Confluence

Macros display

With this connector you can visualize the general information of 1:N projects (size, bugs, vulnerabilities, code smells, code coverage and duplicate code).

Disaggregated information

This application will allow you to add the information of a specific metric to view it. You can also consult the SonarQube or SonarCloud issues of one or more authors.

Critical Data

Thanks to this solution you will be able to view a list of critical and blocking issues (bugs, vulnerabilities and code smells) of a project of your SonarQube or SonarCloud.

Integrate quality into your documentation

Thanks to the SonarQube/SonarCloud connector, you will speed up the process of querying the real status of your code and it will be easier for you to make decisions to improve the status of your code. All without leaving from your Confluence

Atlassian and SonarSource stronger together.

The SonarCloud / SonarQube Connector for Confluence allows access to the most relevant data on the quality of the code in the same space where all the documentation resides.

This integration, thanks to dynamic macros, allows bringing the information from SonarQube / SonarCloud to Confluence, speeding up the consultation process prior to decision-making.

Project Managers will be able to make sure if the quality of the code is good enough directly from Confluence.

What kind of metrics are displayed?

Reliability

Focused on bugs, an issue that represents something wrong in the code.

Security

Focused on vulnerabilities, a security-related issue which represents a potential backdoor for attackers.

Maintainability

Focused on code smells, a maintainability-related issue in the code.

SonarQube Connector for Confluence also allows you to closely study:

  • Duplications Density
  • Lines of code
  • Technical debt and technical debt ratio
  • Coverage
  • Comment density
