Features and benefits of integrating SonarQube with Jira

Discover the main functionalities of the connector between SonarQube Server or SonarQube Cloud and Jira. Improve traceability of your bugs, vulnerabilities, and quality metrics without leaving your Jira instance.

Create Jira tasks from SonarQube issues

Turn issues detected by SonarQube into Jira tasks with just one click. Bugs, code smells, and vulnerabilities are automatically transformed into tasks within your Jira project. Perfect to avoid switching tools — manage everything directly from Jira.



Generate PDF reports from SonarQube results

Export PDF reports with the results of your SonarQube project analysis. You can get five different report types as listed below: a general report offering complete information about your code quality status (reliability, security, coverage, duplications, etc.), and four specialized reports — CWE Top 25, OWASP TOP 10, OWASP ASVS Panel, and ISO 5055.

View code quality directly in Jira

Check code quality metrics, Quality Gate compliance, and detailed issue information visually on your Jira dashboards or boards. No need to switch between tools anymore.

Project Portfolio

Link multiple projects to your Jira project. This feature is especially useful if your Jira project manages several software components and you want to see them all in your SonarQube Server or SonarQube Cloud dashboard.
