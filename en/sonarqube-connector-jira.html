The quality of the code within everyone's reach

All the information on code quality in Jira

Atlassian and SonarSource stronger together.

The SonarCloud / SonarQube Connector for Jira brings your source code quality model to your Jira project, including the quality gate status.

Without leaving Jira, you can see if the project meets or does not meet the quality conditions set in the Quality Gate, including the status of the quality gate and the most important code quality metrics:

  • Reliability
  • Security
  • Maintainability
  • Duplications Density
  • Lines of code
  • Technical debt and technical debt ratio
  • Coverage
  • Comment density

All you need to do is set up the integration between Jira and SonarQube / SonarCloud and you can automatically start creating Jira issues that match your SonarQube bugs, vulnerabilities, and code smells. Quick and simple!

You can also configure several SonarQube / SonarCloud resources to summarize your project portfolio and show a single view of all the metrics.
