Issue Picker Custom Field Type for Jira Cloud

Un nuevo tipo de campo personalizado para buscar y vincular varios problemas al campo.

Incluye: búsqueda rápida de los problemas vistos recientemente, múltiples valores permitidos en un campo y buscador personalizado.
