Installation

{% if page.installation == "jira-installation" %}
  1. Log into your Jira instance as an admin..
  2. Click the admin dropdown and choose Atlassian Marketplace. The Manage add-ons screen loads
  3. Click Find new apps or Find new add-ons from the left-hand side of the page.
  4. Locate the app via search. The appropriate app version appears in the search results.
  5. Click Try free to begin a new trial or Buy now to purchase a license for the app. You're prompted to log into MyAtlassian. The app begins to download.
  6. Enter your information and click Generate license when redirected to MyAtlassian.
  7. Click Apply license.
  8. If you're using an older version of UPM, you can copy and paste the license into your Jira instance.
{% endif %} {% if page.installation == "jira-free-installation" %}
  1. Log into your Jira instance as an admin..
  2. Click the admin dropdown and choose Atlassian Marketplace. The Manage add-ons screen loads
  3. Click Find new apps or Find new add-ons from the left-hand side of the page.
  4. Locate the app via search. The appropriate app version appears in the search results.
  5. Click Install to download and install your app.
  6. You're all set! Click Close in the Installed and ready to go dialog.

{% endif %} {% if page.installation == "sonarqube-installation" %}

The installation is quite simple:

  1. Download and copy the jar file into your extensions/plugins folder in SonarQube installation.
  2. Restart SonarQube server
  3. Log into SonarQube as an admin, go to Administration and General Settings, and then to the plugin section
  4. Paste the license key and save settings
  5. Don't forget to analyze your projects in order process the information with the new plugins.
