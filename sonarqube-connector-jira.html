La calidad del código al alcance de todos

Toda la información de la calidad del código en Jira

Atlassian y SonarSource juntos. SonarCloud / SonarQube Connector para Jira trae tu modelo de calidad de código fuente a tu proyecto de Jira, incluido el estado del umbral de calidad.

Sin salir de Jira, podrás ver si el proyecto cumple o no cumple con las condiciones de calidad establecidas en el umbral de calidad (Quality Gate), así como las métricas de calidad del código más importantes:

  • Fiabilidad
  • Seguridad
  • Mantenibilidad
  • Densidad de duplicaciones
  • Líneas de código (ncloc)
  • Deuda técnica
  • Ratio de deuda técnica
  • Cobertura de código
  • Comentarios Densidad

Todo lo que necesitas hacer es configurar la integración entre Jira y SonarQube / SonarCloud y podrás empezar a crear automáticamente issues de Jira que coincidan con tus bugs, vulnerabilidades y code smells de SonarQube. ¡Rapido y simple!

Además podrás configurar varios recursos de SonarQube / SonarCloud para resumir tu portfolio de proyectos y mostrar una vista única de todas las métricas.
