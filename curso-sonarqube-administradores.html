Curso SonarQube para Administradores

Aprende a instalar, actualizar, configurar y mantener una instancia de SonarQube con herramientas de integración continua para realizar análisis periódicos de código. Este curso se realizará en español y en directo.

Detalles

¿Qué vas a aprender en este curso?

Te desglosamos los objetivos y detalles del Curso de SonarQube para usuarios

Objetivos

  • Conocer la arquitectura de un ecosistema de integración continua
  • Realizar instalaciones y configuraciones complejas en servidores
  • Mantener la instancia actualizada
  • Ejecutar análisis avanzados de código
  • Descubrir la integración con herramientas DevOps como Jenkins, Azure DevOps y Gitlab
  • Administrar un amplio número de proyectos e interpretar la calidad de su conjunto

A quién está dirigido

A SysAdmins, DevOps Engineers, Site Reliability Engineers que quieran aprender a instalar, actualizar, configurar y mantener una instancia de SonarQube con herramientas de integración continua, para realizar análisis periódicos de código.
