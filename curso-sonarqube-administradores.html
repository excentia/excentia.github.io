Curso SonarQube para Administradores

Curso para aprender a instalar, actualizar, configurar y mantener una instancia de SonarQube con herramientas de integración continua para realizar análisis periódicos de código.

El curso de SonarQube para administradores será impartido en español y en directo para que puedas interactuar con el profesor en tiempo real y facilitar tus consultas.

Características

Tutorías Personalizadas

La interacción con el profesor en tiempo real durante el curso para facilitar las consultas y la práctica.

Acreditación Incluida

Detalles

Curso de SonarQube para Administradores en español

Objetivos

  • Conocer la arquitectura de un ecosistema de integración continua
  • Realizar instalaciones y configuraciones complejas en servidores
  • Despliegue de SonarQube en Kubernetes
  • Mantener la instancia actualizada
  • Ejecutar análisis avanzados de código
  • Descubrir la integración con herramientas DevOps como Jenkins, Azure DevOps y Gitlab
  • Administrar un amplio número de proyectos e interpretar la calidad de su conjunto

A quién está dirigido

A SysAdmins, DevOps Engineers, Site Reliability Engineers que quieran profundizar y convertirse en unos verdaderos expertos.
