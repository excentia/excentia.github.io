Issue Picker Custom Field Type for Jira

Un nuevo tipo de campo personalizado para buscar y vincular varios problemas al campo.

Incluye: búsqueda rápida de los problemas vistos recientemente, búsqueda avanzada para seleccionar problemas de los filtros y buscador personalizado.

Prueba por 30 días  

Funcionalidades

Issue Picker Custom Field for Jira

Búsqueda Rápida
de Issues

Al editar los campos creados, estos mostrarán los problemas vistos recientemente para seleccionarlos más rápidamente.

Búsqueda Avanzada
de Issues

Permite seleccionar asuntos a partir de filtros y con selección múltiple.
