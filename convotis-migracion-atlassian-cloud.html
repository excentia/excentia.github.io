VER PDF DEL PROYECTO

CONVOTIS trabajaba sobre una instancia de Jira Software Server y, gracias a su amplia experiencia en entornos de Business Intelligence, obtenía información analítica y operativa clave mediante consultas directas a la base de datos SQL.

Sin embargo, con el lanzamiento de Atlassian Cloud, el acceso directo a la base de datos dejaría de ser posible, lo que obligó a replantear por completo un modelo de extracción de datos que había estado funcionando de manera estable durante años.

Conoce el proyecto de CONVOTIS y excentia

Contexto

CONVOTIS necesitaba migrar su instancia de Server a Cloud y encontrar un nuevo sistema de extracción de datos adaptado a su nueva instancia en la nube.



Objetivo

Encontrar un partner que realizara la migración de su instancia de Jira Software Server a Cloud y establecer un nuevo modelo de extracción de datos.

Solución

CONVOTIS confió en los Servicios de Atlassian Experts de excentia para realizar la migración de su instancia y también para la gestión de las licencias de Jira Cloud.



CONVOTIS es una compañía altamente orientada a la tecnología, con amplia experiencia en Business Intelligence y análisis de datos.

Su fortaleza radica en convertir la información en conocimiento accionable, un pilar que ha sustentado de manera constante todas sus actividades y proyectos estratégicos.





Evolución del proyecto

En excentia, acompañamos a CONVOTIS en su migración a Atlassian Cloud, proporcionando las licencias de Jira necesarias y gestionando la migración de principio a fin. El proyecto incluyó componentes específicos de registro de horas, que requirieron un plan de migración detallado y personalizado. Además, brindamos acompañamiento continuo y soporte técnico para resolver cualquier duda relacionada con el uso de los endpoints de la API REST.



Continuidad analítica

El mayor valor del proyecto reside en la capacidad y profesionalidad del equipo de CONVOTIS, que logró transformar el histórico modelo de BI, basado en consultas SQL a la base de datos de Jira, en un nuevo sistema respaldado por la API REST de Atlassian Cloud. Este cambio garantizó la continuidad de sus capacidades analíticas en el entorno cloud, alineando la plataforma con el futuro de Atlassian sin renunciar a la explotación avanzada de los datos.
